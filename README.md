# Li-garnet

A Python package with utility modules for lithium garnet battery research and simulation.

## Modules Included

- `copy_files.py`: Functions to copy files for workflow automation.
- `delete_subfiles.py`: Script to remove subfiles in a directory.
- `gen_vasp_inputs.py`: Tools to generate VASP input files.
- `Li_partial_occupancy.py`: Utilities for handling partial lithium occupancy.
- `Li9_prime_to_sqs_mo.py`: Functions for converting Li9 prime structures to SQS.

## Installation

pip install li-garnet
